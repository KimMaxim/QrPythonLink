# QrPythonLink
Personal QR-generator for links. Open link at the phone by QR from PC. Cutting all parameters by default

Web_QR.py - simple solution to provide cleared from any ?tags= QR of copied links(from buffer). Help to open browser sources at the phone without any links to your identity and fingerprints when you don't want to link the devices. Have an issue for youtu.be/?video=ABCDEFGH

QR-codegenerator.py can goes with a parametr or provide input for text inserting.

Using MathLib for visualize QR
